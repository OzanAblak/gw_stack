\# Release Notes — v0.1.3-core (2025-11-02, TRT/UTC+3)



\## Özeti

\- Planner minimal ve sağlam Flask app ile yeniden yazıldı.

\- `GET /v1/plan/{planId}` mini handler eklendi (opsiyonel gösterim).

\- Gateway ve lokal SMOKE tek-satır PASS standardına oturtuldu.

\- Paketleme ve etiketleme otomasyonu güncellendi.



\## Değişiklikler

\- `planner/app.py`: health, compile (planId üretir), opsiyonel get handler.

\- `planner/Dockerfile`: python:3.11-slim + waitress, 9090’da servis.

\- `docker-compose.yml`: `planner` build context sade; `gwfwd` ve `gateway` netleştirildi.

\- `infra/gateway/default.conf`: /health ve /v1/ proxy → planner:9090.

\- Betikler: `ci\_smoke\_local.ps1`, `docker/ci\_smoke\_gateway.ps1`, `make\_core\_temp.ps1`, `git\_publish.ps1`.
